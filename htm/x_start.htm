Starting the XAML coding

This section will begin with renaming your XAML file and other first steps.
To rename your file:

  1. In "Solution Explorer" expand your Project (RibbonCompass)
  2. Right click on "MainWindow.xaml"
  3. Select Rename
  4. Type in the highlighted area "RibbonCompass"
For the rest of the set up:
  1. In "Solution Explorer" expand your Project (RibbonCompass)
  2. Double click on "RibbonCompass.xaml"
  3. The XAML and Design Window are now open. In the XAML window is where simple coding is done and the Design window displays what it will look like. Edit the XAML window to read:
Between the "Canvas" class is where one begins to write their code. The "Canvas" class defines an area within which you can explicitly position child objects, using coordinates that are relative to the Canvas area. Important to note: Height="10" Width="100" are the dimensions visible once the instrument is running. Click here for more on the "Canvas" class.
