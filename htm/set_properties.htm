Setting the Properties of your Project

The importance of this step is making sure your Project is in a compatible format with the rest of the solution.

  1. Under the "Solution Explorer" window expand the "RibbonCompass" project
  2. Double click on the "Properties" Icon
  3. Ensure "Target framework:" is selected to ".Net Framework 4 Client Profile"
  4. Ensure "Output type:" is selected to "Class Library"
  5. Select the "Build" Pane
  6. Change "Platform target" to "x86"
  7. Save
