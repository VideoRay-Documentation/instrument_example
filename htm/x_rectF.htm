Fill Property and ImageBrush Class

Fill is a property that gets or sets the Brush that specifies the shape’s ( used with the rectangle class in the example) interior. A color is therefore a valid value for the fill property. ImageBrush is also a valid value for the fill property. ImageBrush takes an image and paints it on the specific shape. For more information on the Fill property click here. For more information on the ImageBrush class click here.

ImageSource

ImageSource specifies where the image is coming from for the ImageBrush. The path of the image can be from any folder on the computer, with the correct path. Without a "C:" or likewise prefix, the folder path assumes that it starts at the Project’s name’s folder. So in the example’s case "InstrumentHostSample". For information on the ImageSource Property click here.

How to Make an Image a Resource

The examples makes a .png file a resource, so it can be referenced from any computer and not have to be hard coded. For the example, an image made in paint was used as the image to be brushed onto the rectangle. The png file was of made from 24 equal parts each to represent 45 degrees. 45*24= 1080. The width of the rectangle was conveniently set to 1080, the width for viewing was set to 360 (Before the first <Grid> or <Canvas> is invoked) and the rectangle was set to the right by 417 so that the examples image would originally set with North in the viewable center. To add a .png or .jpg file as a resource:

  1. Expand your Project (RibbonCompass) in "Solution Explorer"
  2. Right click on "Resource"
  3. Select "Add"
  4. Select "Existing Item… "
  5. Navigate and select your image (you may have to change the type of files which can be selected)
  6. Click "Add"
  7. Expand the "Resources" folder in the "Solution Explorer" within your expanded project
  8. Select "Your image" by left clicking on it
  9. In the "Properties" Window (View> Properties if you cannot see it) select "Resource" from the dropdown menu for "Build Action"
