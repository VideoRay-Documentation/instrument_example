The Rectangle Class

To draw a Rectangle call on the rectangle class. The Ribbon Compass example utilizes the Rectangle class in the following manner: In the Ribbon Compass example x:Name="Ribbon5" sets the name of the rectangle. Names are for referencing the class in later transforms and other manipulations in XAML and XAML.cs (C#) file. Width="" sets the rectangle’s width which is very important. The ImageBrush section describes how the width of the rectangle comes into play for the overall Instrument. Height="" sets the rectangles height. Canvas.Left="-417"sets the position of the rectangle to the negative left (so right) of the canvas. Canvas.Top="-10"sets the position of the rectangle to the negative top (so bottom) of the canvas. For information on the rectangle class click here. The properties of Rectangle.Fill and Rectangle.Transform are covered in there own sections.
