The Rectangle Class

To draw a Rectangle call on the rectangle class. The Ribbon Compass example utilizes the Rectangle class in the following manner: In the Ribbon Compass example x:Name="Ribbon5" sets the name of the rectangle. Width="" sets the rectangle’s width which is very important. "ImageBrush" in its own. Height="" sets the rectangles height. Canvas.Left="-417"sets the position of the rectangle to the negative left (so right) of the canvas. Canvas.Top="-10"sets the position of the rectangle to the negative top (so bottom) of the canvas.
