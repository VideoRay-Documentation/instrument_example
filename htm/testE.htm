Testing Environment

These procedures describe how to get the RibbonCompass project running within the "vrInstrumentHostSample" test solution and how to edit testable parameters. This Solution was provided as a simple testing simulation. Its current configuration is set to alter just the Yaw parameter, but it can be easily modified to change a breath of settings to mimic the real system.

Add the RibbonCompass "Project" as a "reference"

This step describes how to make sure that the project RibbonCompass is being referenced by the vrInstrumentHostSample Solution.

  1. Navigate to the "Solution Explorer"
  2. Expand the "vrInstrumentHostSample" Project
  3. Right click on "References"
  4. Select "Add Reference…"
  5. On the left pane click "Solution"
  6. Click on the name of your project (in this case "RibbonCompass")
  7. Ensure the a check mark is in the box
  8. Click on "OK"
