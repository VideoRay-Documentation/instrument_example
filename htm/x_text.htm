The TextBlock Class

The TextBlock class provides a lightweight control for displaying small amounts of flow content. Some of the properties used by the TextBlock have been already discussed in prior sections. Click here to learn about the Fill property and the Opacity property. Click here to learn about the Width, Height and Left properties . HorizontalAlignment="" can be set to left, right center and is similar to Microsoft word's left, right center functions. VerticalAlignment="" is similar to HorizontalAlignment="", only it is controlling a different axis for the text. TextWrapping="" allows for the possibility of wrapped text. Foreground="" sets the color of the foreground. Margin="" sets the dead area around the TextBlock. FontSize="" Controls the actual size of the text. Text="" is what the actual text reads as.
