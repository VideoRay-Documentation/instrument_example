Open the "vrInstrumentHostSample" Solution

In Microsoft Visual Studio

  1. Click "file"
  2. Select "Open"
  3. Click "Project/Solution"
  4. Browse to where you put the downloaded file (example would be C: \Documents\Visual Studio 2013\Projects)
  5. Continue to browse to the "InstrumentHostSample" folder
  6. Select "vrInstrumentHostSample" solution
  7. Click "Open"

OR
  1. Navigate through your files to where you downloaded file (example would be C: \Documents\Visual Studio 2013\Projects)
  2. Continue to browse to the "InstrumentHostSample" folder
  3. Double click the "vrInstrumentHostSample" solution
