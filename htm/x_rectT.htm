Rectangle RenderTransform & TranslateTransform

RenderTransform gets or sets transform information that affects the rendering position of this element. This is a dependency property. TranslateTransform sets the distance to which to move the object (in the example’s case a rectangle) along an axis (in the example’s case X). The x:Name="RibbonPost" allows the C# files to interact with it since it has a name. The X="0" initially sets the X axis displacement to 0, but the C# code will interact and change that. If you wanted to move something along the Y axis you would just change the X to a Y. For more information on the RenderTransform Property click here.
