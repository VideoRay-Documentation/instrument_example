Create a new Project within the Solution

In Microsoft Visual Studio:

  1. Click "file"
  2. Select "Add"
  3. Select "New Project…"
  4. In the "Add New Project" window select "Installed" on the left pane
  5. Next select "Visual C#" in the same left pane
  6. In the middle pane click "WPF Application"
  7. Above the middle pane, at the top, navigate and set the drop down menu to ".NET Framework 4"
  8. At the bottom of the window change the "Name" to "RibbonCompass"
  9. At the bottom of the window change "Location" to the folder to be in the "InstrumentHostSample" folder (example would be C:\...\Documents\Visual Studio 2013\Projects\ InstrumentHostSample"
  10. Click "Ok"
