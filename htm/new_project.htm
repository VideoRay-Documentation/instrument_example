Create a new Project within the Solution

In Microsoft Visual Studio:

  1. Click "file"
  2. Select "Add"
  3. Select "New Project…"
  4. In the "Add New Project" window select "Installed" on the left pane
  5. Next select "Visual C#" in the same left pane
  6. In the middle pane click "WPF Application"
  7. At the bottom of the window change the "Name" to "Your Instrument's name" (example Ribbon Compass)
  8. At the bottom of the window change "Location" to the folder to be in the "SDK_Sample_CockpitInstrumentHost" folder (example would be C:\...\Documents\Visual Studio\Projects\ SDK_Sample_CockpitInstrumentHost"
  9. Click "Ok"
