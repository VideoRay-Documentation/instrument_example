The Rectangle Class

To draw a Rectangle use the rectangle class. Width="" and Height="" set the width and height of the rectangle. The property x:Name="" sets the name of the rectangle. Names are for referencing the specific class (in this case a rectangle) in transforms and other manipulations in XAML or XAML.cs (C#) files. Width="" sets the rectangle’s width which is very important for the Ribbon Compass. The ImageBrush section describes how the width of the rectangle comes into play for the overall Ribbon Compass Instrument. Height="" sets the value of rectangle's height. Canvas.Left="" sets the the value of displacement between that the left of the rectangle and the left of the canvas (along the X axis). Canvas.Top="" sets the position of the top of the rectangle off the top of the Canvas along the Y axis. If the value is set to a negative, the shape's top will be negatively displaced from the top of the Canvas. For more information on the rectangle class click here.
