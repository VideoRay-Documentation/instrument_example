How to Integrate New Instruments into VideoRay Cockpit

To put the new instrument into Cockpit, you will need a flash-drive or some other way (email) to get the .dll file from your local computer to the integrated control box's laptop. The .dll is all you will need to have your instrument running. There are a couple of locations where this can be found. For the Ribbon Compass example:

  1. Open Windows Explorer
  2. Navigate to the "IntrumentHostSample" folder
  3. Continue to navigate along the path "\Ribbon\obj\Debug"
  4. Copy the RibbonCopmass.dll file onto your flash drive

Regardless if using email or a flash drive, just get the .dll file to the integrated control box's laptop. Once the .dll file is on the laptop:
  1. Open the laptop's Windows Explorer
  2. Navigate to the "C:\ProgramFiles(x86)\VideoRay\vrCockpit\Instruments" folder
  3. Copy the RibbonCopmass.dll file into this location
  4. Close Windows Explorer
  5. Run Cockpit
At this point the instrument should be running. A common mistake is setting the size of the instrument too small so that it is hard to see. The size of the instrument can be altered in either its xaml or xaml.cs file. Another area to try diagnosing if the the instrument does not show is to go to the Instruments Display and see if the instrument is unselected.
